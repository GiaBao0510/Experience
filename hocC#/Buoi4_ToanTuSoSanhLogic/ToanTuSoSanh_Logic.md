## **Buổi 4:** Toán tử so sánh, logic

Các toán tử so sánh:
- <i>==</i> :toán tử so sánh bằng.
- <i>> :</i>toán tử so sánh lớn hơn.
- < :toán tử so sánh nhỏ hơn.
- >= : toán tử so sánh lớn hơn hoặc bằng. 
- <= : toán tử so sánh nhỏ hơn hoặc bằng.
- != : toán tử so sánh khác.

Các phép toán logic:
- &&: Phép toán **và***.
- || : Phép toán **hoặc**.
- ! : Phép toán **phủ định**.
